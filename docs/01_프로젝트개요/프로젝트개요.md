# ClassOn 프로젝트 개요

## 프로그램 정보
- **프로그램명**: ClassOn
- **목적**: 반도체 설계 등 취업 실무 교육 강사를 위한 교실 운영 통합 플랫폼
- **버전**: v1.0 (초기 설계)
- **설계일**: 2026-03-11

---

## 배경 및 문제 정의

### 강사의 고민
- 오프라인 교실(30~50명)에서 학생들이 수업을 잘 따라오는지 파악이 어려움
- 학생들이 너무 조용하고 소극적 → 피드백을 받기 어려움
- 3개월(하루 7시간) 집중 과정에서 학생 한 명 한 명을 케어해야 하나 경험과 기억에만 의존
- 팀 편성 시 학생 실력 파악이 주관적

### 해결 방향
- 실시간 익명 피드백 시스템으로 솔직한 학생 반응 수집
- 강사 화면에 교실 레이아웃을 시각화하여 한눈에 현황 파악
- 데이터 기반 학생 성취도 관리 및 팀 편성

---

## 강의 환경
- **수업 형태**: 오프라인 (한 교실)
- **수강 인원**: 교실마다 다름, 약 30~50명
- **장비**: 학생 개인 PC (반도체 설계 실습용), 같은 네트워크 연결
- **수업 기간**: 3개월, 하루 7시간
- **과정 종류**: 반도체 설계, 임베디드 등 여러 과정
- **하루 수업**: 1개 과정만 진행
- **강사**: 1명 (강사 PC에서 서버 실행)

## 사용자 구분
| 사용자 | 접근 방법 | 권한 |
|--------|----------|------|
| 강사 | ID/PW 로그인 | 전체 기능 |
| 학생 | 이름 클릭 입장 | 피드백 화면만 |

---

## 실행 환경
- **OS**: Windows
- **설치 필요**: Node.js (1회)
- **실행**: 배치파일 더블클릭
- **접속**: 같은 네트워크 브라우저 (인터넷 불필요)
- **데이터**: 로컬 SQLite DB 파일 1개
