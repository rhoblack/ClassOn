# ClassOn 화면 설계서

## 화면 목록

| 화면 | URL | 접근 권한 |
|------|-----|---------|
| 로그인 | /login | 누구나 |
| 메인 대시보드 | /teacher/dashboard | 강사 |
| 수업 진행 | /teacher/class | 강사 |
| 학생 관리 | /teacher/students | 강사 |
| 퀴즈 출제 | /teacher/quiz | 강사 |
| 문제 은행 | /teacher/questions | 강사 |
| 팀 편성 | /teacher/teams | 강사 |
| 시스템 설정 | /teacher/settings | 강사 |
| 학생 입장 | /student | 학생 |
| 학생 피드백 | /student/class | 학생 |
