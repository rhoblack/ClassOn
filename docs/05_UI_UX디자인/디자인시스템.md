# ClassOn 디자인 시스템

## 브랜드

### 프로그램명
- **ClassOn** — "수업이 켜진다 (Class + On)"

### 로고
- 스타일: 텍스트형 로고 (아이콘 없음)
- 구성: "Class" (검정) + "On" (파랑 #2563EB)
- 심플하고 직관적인 표현

## 디자인 원칙
