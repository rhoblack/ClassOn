# ClassOn 기능 명세서

## 모듈 1 — 과정(코호트) 관리

| 기능 | 설명 | 비고 |
|------|------|------|
| 과정 생성 | 과정명, 기수, 시작일, 종료일 입력 | 예: 반도체설계 3기 |
| 과정 수정 | 과정 정보 수정 | |
| 과정 종료 | 과정을 종료 상태로 변경 | 데이터 보존 |
| 하루 1개 과정 활성화 | 당일 수업할 과정 선택 | |
| 과정별 데이터 분리 | 학생/퀴즈/이력 독립 관리 | 과정간 데이터 혼용 없음 |

## 모듈 2 — 실시간 수업 피드백

### 학생 측
| 기능 | 설명 |
|------|------|
| 이해도 버튼 클릭 | 커스텀 세트 버튼 클릭으로 상태 전송 |
| 익명 질문 입력 | 텍스트 입력, 강사에게만 표시 |
| 실습 도움 요청 | 도움 요청 버튼 클릭 → 대기열 등록 |

### 강사 측
| 기능 | 설명 |
|------|------|
| 책상 배치도 실시간 현황 | 책상 색상으로 학생 상태 표시 |
| 전체 현황 집계 | 상태별 인원수, 비율 표시 |
| 질문 모아보기 | 익명 질문 목록, 확인 처리 |
| 도움 요청 대기열 | 요청 순서대로 목록 표시 |
| 수업 모드 전환 | 설명 / 실습 / 퀴즈 모드 |
| 전체 공지 푸시 | 텍스트 + 링크 (새탭 오픈) |

### 피드백 버튼 커스터마이징
| 항목 | 내용 |
|------|------|
| 버튼 세트 | 미리 저장, 수업 중 전환 |
| 버튼 개수 | 2~5개 |
| 스타일 | 이모지 or 색상버튼 선택 |
| 색상 | 16색 팔레트 (진한 8 + 연한 8) |
| 이모지 카테고리 | 감정(16) / 손동작(16) / 기호(16) / 동물(8) / 물건(8) |
| 책상 색상 연동 | 버튼 색상 = 책상 표시 색상 |
| 기본 세트 | ① 이해도확인 ② 실습진행도 ③ 속도조절 |

**기본 세트 상세:**
```
① 이해도 확인: 😊잘됨(초록) 🤔애매(노랑) 😵모름(빨강) ✋질문(파랑)
② 실습 진행도: ✅완료(초록) 🔄진행중(노랑) 🆘막힘(빨강)
③ 속도 조절:  🐇빨라요(파랑) 👍적당해요(초록) 🐢느려요(노랑)
```

**책상 상태 색상 (기본 세트 기준):**
```
🟢 #10B981  잘됨
🟡 #F59E0B  애매함
🔴 #EF4444  모름
🔵 #2563EB  질문
⚫ #94A3B8  미응답
```

## 모듈 3 — 퀴즈 & 성취도

### 퀴즈 출제
| 기능 | 설명 |
|------|------|
| 즉석 출제 탭 | 그자리에서 문제 직접 입력 후 출제 |
| 문제은행 출제 탭 | 과목/챕터 선택 후 랜덤 or 직접 선택 |
| OX 퀴즈 | 즉석 개념 확인 |
| 4지선다 퀴즈 | 문제/보기 4개/정답/제한시간 설정 |
| 즉석 문제 저장 | 출제 후 문제은행에 저장 가능 |
| 제한시간 | 초 단위 설정 |

### 퀴즈 진행
| 기능 | 설명 |
|------|------|
| 실시간 응답 집계 | 선택지별 인원수, 비율 즉시 표시 |
| 오답 분석 | 많이 틀린 보기 하이라이트 |
| 응답 현황 | X/전체명 응답 완료 표시 |

### 성취도 관리
| 기능 | 설명 |
|------|------|
| 성취도 계산 | 퀴즈 70% + 이해도 30% |
| 학생별 누적 기록 | 퀴즈 정답률, 성장 곡선 |
| 등수 표시 | 강사 화면에만 (학생 비공개) |
| 단원별 취약점 분석 | 챕터별 전체 정답률 |

## 모듈 4 — 학생 관리

| 기능 | 설명 |
|------|------|
| 학생 직접 입력 | 이름/성별/나이/학교/전공/전화번호 |
| 엑셀/CSV 업로드 | 컬럼 매핑 지원, 미리보기 확인 후 확정 |
| 과정별 학생 할당 | 과정마다 독립된 학생 목록 |
| 접속 방식 | 목록에서 이름 클릭으로 입장 |
| 출석 자동 체크 | 접속=출석, 미접속=결석 자동 처리 |
| 책상 클릭 팝업 | 프로필+성취도+이해도이력+출석률+강사메모 |
| 강사 메모 | 특성, 관찰 사항 자유 입력 |
| 성취도 목록 | 성취도순/이름순/이해도순 정렬 |

**학생 프로필 항목:**
```
이름, 성별, 나이, 학교, 전공, 전화번호,
성취도(%), 등수, 출석률,
이해도 추이 (최근 기록),
강사 메모
```

## 모듈 5 — 문제 은행

| 기능 | 설명 |
|------|------|
| 과목 관리 | 과목 추가/수정/삭제 |
| 챕터 관리 | 챕터 추가/수정/삭제 |
| 문제 직접 입력 | 과목/챕터/유형/문제/보기/정답/해설 |

| 직접 선택 출제 | 개별 문제 체크 후 출제 |
| 과정별 분리 | 과정마다 독립된 문제 은행 |

**엑셀 양식 컬럼:**
```
과목 / 챕터 / 유형(OX,4지) / 문제 / 보기A / 보기B / 보기C / 보기D / 정답 / 해설
```

## 모듈 8 — 시스템 운영

| 기능 | 설명 |
|------|------|
| 배치파일 실행 | ClassOn_시작.bat → 서버 실행 + 브라우저 자동 오픈 |
